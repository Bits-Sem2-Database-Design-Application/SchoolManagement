Welcome to School Management System !!!

This is self service portal created for new applicants, students and school staff to access and manage schools related information and operations. It allows users to perform following operations:

A. New Student Applicant

  • Application to enroll for a Program
  • Check application status
  • Register login user on enrolling

B. Students

  • Login to Portal
  • Forgot Password
  • Change Password
  • Post Change Request
  • View 'Open' and 'Closed' Change Requests
  • Enter Fees Transaction Details
  • Download Fees Receipt
  • View Performance Report

C. Admin Staff

  • View, Approve or Reject Student Application
  • Add New Enrollment
  • View and Edit Student Profile
  • Generate Batch-wise Student Report
  • Update Student Fee Details
  • View Report: Pending Fees
  • View Report: Fees Transactions
  • Update Student Marks
  • Batch-wise Report: Student Pass or Failed
